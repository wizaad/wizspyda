# WizSpyda

**Version 1.0.0**

A simple program to search through the most recent questions on stackoverflow and their vote count.

---

## Installation

git clone https://github.com/wizaad/wizspyda.git

pip3 install -r requirements.txt 

---

## Usage

python3 main.py
